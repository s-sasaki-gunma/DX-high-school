# DX-high-school
raspberry pi 5の設定等で用いたスクリプト郡
ご自身の責任においてご利用ください

## setProxy.sh
群馬県の県立高校の教育用ネットワークにおいてRaspberry pi 5を使う場合のProxyの設定スクリプト

## setApt.sh
群馬県の県立高校の教育用ネットワークにおいてRaspberry pi 5を使う場合のAptの設定スクリプト

## setNTP.sh
Raspberry pi 5のNTPサーバ有効化スクリプト
- NTPサーバのローカルIPのデフォルト値："192.168.0.2"
- 必要に応じてスクリプトを書き換えてから実行してください

## setVNC.sh
Raspberry pi 5のVNC有効化スクリプト
- パスワードデフォルト値："raspberry"
- 必要に応じてスクリプトを書き換えてから実行してください

## setAccessPoint.sh
Raspberry pi 5をアクセスポイント化するスクリプト
- APとしてのIPアドレスデフォルト値："192．168．111．1"
- APとしてのパスワードデフォルト値："12345678"
- 必要に応じてスクリプトを書き換えてから実行してください
